# Creden

**Inscrição e entrada sem complicação.**

Plataforma web (Next.js) para organizadores de eventos criarem páginas de inscrição,
gerenciarem participantes e fazerem check-in via QR code no dia do evento. O participante
se inscreve, recebe um ingresso digital com QR code por e-mail e apresenta na entrada.
